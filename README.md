# Adbmal Calculus Formalization

This is an archive of the formalization work on Adbmal (mirrored lambda)
by Dimitri Hendriks and Vincent van Oostrom. I adapted the proofs to
work with modern Rocq.

Based on extractions of the files from
https://web.archive.org/web/20050209112819/http://www.cs.ru.nl/~hendriks/coq/adbmal/
and
https://web.archive.org/web/20070227003639id\_/http://www.phil.uu.nl:80/preprints/lgpr/001/adbmal.tar.gz
